==============================================================================
INTRODUCTION                                            *tmuxlike-intro*

vim-tmuxlike provides tmux-style prefix commands for navigating, splitting,
resizing, and managing editor windows and tabs.

The default prefix is CTRL-A. Every command below starts with the prefix.
Neovim's Lua features require Neovim 0.11 or later.
Vim's interactive resize mode and built-in chooser require +popupwin.

==============================================================================
COMMANDS                                                *tmuxlike-commands*

WINDOWS

 h j k l, arrow keys   Move to a neighboring window.
 ;                    Move to the previously selected window.
 z                    Toggle zoom for the current window.
 %                    Create a vertical split with a new buffer.
 |                    Vertically split the current buffer.
 "                    Create a horizontal split with a new buffer.
 _                    Horizontally split the current buffer.
 x                    Confirm and close the current window.
 !                    Move the current file to a new tab.
 Space                Cycle through the available pane layouts.
 {                    Swap with the previous pane.
 }                    Swap with the next pane.

The layout sequence is even-horizontal, even-vertical, main-horizontal, and
main-vertical. Pane swapping wraps around, and focus follows the original pane
content.

RESIZE MODE

 H                    Increase the current window's width.
 L                    Decrease the current window's width.
 J                    Increase the current window's height.
 K                    Decrease the current window's height.

Press Escape, Enter, or q to leave resize mode. Other keys are ignored while
the mode is active.

TABS

 c                    Open a new tab after the last tab.
 CTRL-H, CTRL-P       Select the previous tab.
 CTRL-N, CTRL-L       Select the next tab.
 &                    Confirm and close the current tab.

OTHER

 ?                    Open this help page.
 ~                    Show message history.
 d                    Suspend Vim or Neovim.
 r                    Redraw the screen.
 t                    Display the current time.
 ]                    Paste from the + register.
 q s =                Open the window chooser.

Vim and Neovim include a built-in window chooser for the q, s, and = mappings.
When multiple tabs are open, the chooser also displays a numbered tab selector.
Press 1 through 9 to jump directly to a tab.
Pane labels can be selected with either lowercase or uppercase letters.
Their glyphs are generated from TOIlet. The available fonts are "smblock"
(default) and "pagga". Pane labels use the preset sequence
ABEFHIJKLMNOPRSTUVWXYZ, excluding C, D, G, and Q.

==============================================================================
CONFIGURATION                                             *tmuxlike-config*

Change the prefix by mapping to <Plug>(tmuxlike-prefix): >

    nmap <C-\> <Plug>(tmuxlike-prefix)
    nmap <Leader><Leader> <Plug>(tmuxlike-prefix)
<
Change the current-buffer split keys before the plugin is loaded: >

    let g:tmuxlike_key_vsplit = '\'
    let g:tmuxlike_key_hsplit = '-'
<
Configure message history: >

    let g:tmuxlike_messages_container = 'scratch'
<
Supported values are "scratch", "float", and any other value for the standard
:messages display.

Configure Vim's built-in window chooser before loading the plugin: >

    let g:tmuxlike_chooser_scope = 'current'
    let g:tmuxlike_chooser_font = 'smblock'
    let g:tmuxlike_chooser_characters = 'ABEFHIJKLMNOPRSTUVWXYZ'
<
The scope may be "current" or "all".

==============================================================================
NEOVIM LUA SETUP                                             *tmuxlike-lua*

Neovim users can configure all command keys with require("tmuxlike").setup(): >

    lua << EOF
    require("tmuxlike").setup({
      prefix = "<C-a>",
      resize_step = 3,
      messages = "float",
      chooser = {
        scope = "current",
        font = "smblock",
        characters = "ABEFHIJKLMNOPRSTUVWXYZ",
      },
      mappings = {
        messages = "~",
        choose_window = { "q", "s", "=" },
        split_vertical = "|",
        split_horizontal = "_",
        next_layout = "<Space>",
        swap_pane_previous = "{",
        swap_pane_next = "}",
      },
    })
    EOF
<
Set a mapping to false to disable it. A mapping may also be a list of keys.
The chooser scope may be "current" for window labels in the current tab, or
"all" for a list containing windows from every tab.
The labels c, d, g, q, and digits are reserved and automatically excluded.
Digits 1 through 9 select tabs when multiple tabs are open.
Explicitly configuring C, D, G, or Q raises an error.
